## Requisitos
- PHP 7.4.3
- Apache 2.4.41
- MySQL Server 8.0.28
- Biblioteca php-mbstring
- É recomendado que o servidor esteja configurado para o padrão UTF-8.
- É recomendado que o conector php-mysql seja instalado manualmente em sistemas Linux/Unix.
